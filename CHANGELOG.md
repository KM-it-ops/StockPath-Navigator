# Changelog

All notable changes to StockPath Navigator are documented here.

## [3.0.0] - 2026-02-19

### Added
- **Proactive Conditional Trade Plans** — "If/Then" action plans with bullish, bearish, and sideways scenarios per stock. Maximum 5 active plans with expiry dates.
- **Weekly Monday Briefing** — Comprehensive weekly market prep report including macro outlook, key economic events by day, sector heat map, 5-stock watchlist with conditional plans, open position checkups, and "Lesson of the Week."
- **Emotional Intelligence Protocol** — Detects 4 emotional states (frustration, anxiety, euphoria, defeat) via language pattern analysis. Each state triggers a tailored adaptive response framework.
- **Emotional Circuit Breaker** — Auto-activates on 3+ emotional signals in a single message, pausing trade discussion and grounding the user in objective portfolio metrics.
- **Interactive React Dashboard** — Companion UI with 5 tabs: Overview, Trade Journal, Conditional Plans, Mindset/Emotional Health, and Goal Tracker.
- Response protocol expanded from 6 to 8 steps (added emotional check and conditional plan generation).
- Dashboard triggers for automatic display (8 conditions).
- Monday Briefing search checklist (7 mandatory searches).

### Changed
- Core identity now includes emotional intelligence as explicit competency.
- Prohibited behaviors expanded to include ignoring emotional signals and matching user's emotional intensity.
- First message protocol now introduces proactive alerts and weekly briefings.

## [2.0.0] - 2026-02-19

### Added
- **Real-Time Data Integration** — Full web search protocol with 6-step mandatory search workflow before every recommendation. Data freshness rules, cross-verification, and source citation.
- **Multi-Strategy Phasing** — 4 distinct strategies (Swing, Momentum, Options, Dividend) gated behind experience and capital milestones with explicit graduation criteria.
- **Trade Journal System** — Persistent trade logging with entry/exit records, P/L tracking, strategy classification, and lesson extraction.
- **Visual Portfolio Dashboard** — 5 ASCII components: portfolio overview, open positions table, allocation bar chart, performance metrics, and goal progress tracker.
- **Dashboard Triggers** — 8 automatic conditions for dashboard display.

### Changed
- Response protocol now includes live data verification at every step.
- Entry thesis now includes data timestamps and source citations.
- Strategy framework now specifies exact allocation percentages per phase.

## [1.0.0] - 2026-02-19

### Added
- Core analyst persona with fiduciary mindset.
- 6-step mandatory reasoning chain (market context → sector scan → stock screening → entry thesis → self-audit → plain-language summary).
- 4-phase capital growth framework ($500 → $100K+).
- Structured output format with watchlist template.
- Hallucination guardrails and mandatory disclaimers.
- Teaching integration with progressive explanation depth.
- Prohibited behaviors list.
- First message onboarding protocol with honest math.
